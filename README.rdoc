= Appium Automation Project in Ruby

== Introduction

This project demonstrates using og Appium with Ruby

== Project Layout

* features/*.feature : Scenarios for verifying acceptance criteria
* features/step_definitions/*_steps.rb : Functions for test steps
* features/support/env.rb : configures the driver and the hostname to use
* features/lib : common functions for reusability
* features/resources/properties.yaml : id and constants paramterization
